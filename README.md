# 🌿 Meet A Flora

A Flutter application that identifies plants using images and provides dynamic, AI-generated botanical information.

---

## 📱 Project Overview

**Meet A Flora** is a smart plant recognition app that allows users to:

- Capture a plant using the camera 📷
- Select an image from the gallery 🖼️
- Get **real-time AI-generated plant details** 🌱

The app follows **Clean Architecture** and uses modern Flutter development practices.

---

## 🚀 Features

- 📸 Image Picker (Camera & Gallery)
- 🤖 AI Plant Recognition (via API)
- 🔄 Dynamic Data (fresh info every time)
- 🧠 Clean Architecture
- ⚡ State Management using Cubit (BLoC)
- 🌐 API Integration using Dio
- 🧭 Navigation using GoRouter

---

## 🧩 App Flow

### 1️⃣ Discovery Screen
- Capture image OR select from gallery
- Navigate to details screen

### 2️⃣ Plant Details Screen
- Sends image to API
- Displays:
  - 🌿 Plant Name
  - 📖 Description
  - ☀️ Sunlight Needs
  - 🌱 Care Tips
